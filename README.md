# Classymouse Kodi Addon Repository

## Install in Kodi

1. In Kodi go to **Settings → File Manager → Add source**
2. Enter URL: `https://classymouse.github.io/repository.classymouse/`
3. Give it a name, e.g. `classymouse`
4. Go to **Add-ons → Install from zip file** → pick the source → install the repo zip
5. Then install individual addons from **Install from repository → Classymouse Repository**
